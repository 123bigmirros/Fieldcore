# OpenManus Frontend

OpenManus机器移动可视化前端界面

## 安装依赖

```bash
npm install
```

## 启动开发服务器

```bash
npm run dev
```

前端将在 http://localhost:3000 启动

## 构建生产版本

```bash
npm run build
```

## 预览生产版本

```bash
npm run preview
```

## 功能特性

- 实时显示所有机器的位置和状态
- 3D坐标映射到2D可视化界面
- 自动刷新功能
- 详细的机器信息面板
- 响应式设计

## 使用说明

1. 确保后端API服务器正在运行 (端口8000)
2. 启动前端开发服务器
3. 在浏览器中访问 http://localhost:3000
4. 点击"自动刷新"按钮开始实时监控
5. 在右侧面板查看详细的机器信息
